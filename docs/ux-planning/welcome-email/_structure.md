# Structure — Welkomstmail

## Sectievolgorde

1. Begroeting + openingszin (warmte, je bent welkom)
2. Visie (waarom bestaat dit — het warme water samen uitvinden)
3. Hoe het werkt (initiatieven → fiches, één alinea)
4. Stat ter verankering
5. Wat je kunt doen (bookmarken, reageren — kort)
6. CTA: Ontdek initiatieven
7. Aankondiging onboardingserie
8. Afsluiting

## Motivatie volgorde

Visie staat vóór uitleg van het model: mensen kopen eerst het *waarom* voor ze in de *hoe* duiken. De stat staat ná de uitleg — "X fiches" heeft pas betekenis als je weet wat een fiche is.
